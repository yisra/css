טרנספורמציות (Transformations)

הזזה (Translate)

מאפיין דוגמה הסבר
translate
transform: translate(50px, 100px);
 מזיז את האלמנט על ציר ה-X וה-Y

סיבוב (Rotate)

מאפיין דוגמה הסבר
rotate
transform: rotate(45deg);
 מסובב את האלמנט לפי זווית

שינוי גודל (Scale)

מאפיין דוגמה הסבר
scale
transform: scale(1.5);
 משנה את גודל האלמנט (יחסית)

הטיה (Skew)

מאפיין דוגמה הסבר
skew
transform: skew(30deg, 20deg);
 מטה את האלמנט על צירי ה-X וה-Y

מטריצה (Matrix)

מאפיין דוגמה הסבר
matrix
transform: matrix(1, 0, 0, 1, 50, 100);
 משלב הזזה, סיבוב, שינוי גודל והטיה במאפיין אחד
