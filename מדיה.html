מדיה (Media Queries)

מדיה ב-CSS מאפשרת להתאים סגנונות בהתאם למאפייני מכשירים כמו גודל מסך, רזולוציה ועוד.

תחביר (Syntax)

תחביר בסיסי של Media Query:

@media (max-width: 600px) { body { background-color: lightblue; } }

בדוגמה זו, הרקע ישתנה לכחול בהיר במסכים שגודלם קטן מ-600px.

דוגמאות שימושיות

דוגמה הסבר
@media (min-width: 768px) { body { font-size: 18px; } }
מתאים את גודל הפונט למסכים ברוחב של 768px ומעלה.
@media (orientation: landscape) { body { background-color: lightgreen; } }
משנה את הרקע כאשר המסך במצב לרוחב.

מאפייני מדיה

להלן מאפיינים נפוצים לשימוש ב-Media Queries:

מאפיין תיאור
max-width רוחב מקסימלי של המסך
min-width רוחב מינימלי של המסך
orientation מצב המסך (אנכי/אופקי)
resolution רזולוציית המסך
