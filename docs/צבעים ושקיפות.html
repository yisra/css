צבעים (Colors)

שמות צבעים

ניתן להשתמש בשמות צבעים מוגדרים מראש כמו red, blue, או green.

color: red;

קודי HEX

קוד HEX הוא ייצוג הקסדצימלי של צבע, כמו #ff0000 לאדום.

color: #ff0000;

צבעי RGB

RGB מאפשר לקבוע צבע באמצעות ערכי אדום, ירוק וכחול.

color: rgb(255, 0, 0);

צבעי HSL

HSL מייצג גוון (Hue), רוויה (Saturation) ובהירות (Lightness).

color: hsl(0, 100%, 50%);

שקיפות (Opacity)

RGBA

RGBA דומה ל-RGB, אך כולל ערך שקיפות (Alpha) הנע בין 0 ל-1.

background-color: rgba(255, 0, 0, 0.5);

מאפיין השקיפות (Opacity)

מאפיין opacity משפיע על כל האלמנט, כולל התוכן שבו.

opacity: 0.5;
