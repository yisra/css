גריד (Grid)

הגדרת מיכל גריד (Grid Container)

מאפיין דוגמה הסבר
display: grid
display: grid;
 הגדרת אלמנט כמיכל גריד
display: inline-grid
display: inline-grid;
 גריד במצב אינליין

מאפייני פריטי גריד (Grid Items)

מאפיין דוגמה הסבר
grid-column
grid-column: 1 / 3;
 קובע את העמודות בהן הפריט יתפרס
grid-row
grid-row: 2 / 4;
 קובע את השורות בהן הפריט יתפרס

תבנית גריד (Grid Template)

מאפיין דוגמה הסבר
grid-template-columns
grid-template-columns: 1fr 2fr;
 קובע את מבנה העמודות
grid-template-rows
grid-template-rows: 100px auto;
 קובע את מבנה השורות

רווח בין פריטים (Grid Gap)

מאפיין דוגמה הסבר
gap
gap: 10px;
 קובע את הרווח בין הפריטים
row-gap
row-gap: 15px;
 רווח בין שורות
column-gap
column-gap: 20px;
 רווח בין עמודות

פלקסבוקס (Flexbox)

הגדרת מיכל פלקסבוקס (Flex Container)

מאפיין דוגמה הסבר
display: flex
display: flex;
 הגדרת אלמנט כמיכל פלקסבוקס
flex-direction
flex-direction: row;
 קובע את כיוון הפריטים במיכל
justify-content
justify-content: center;
 יישור הפריטים על הציר האופקי
align-items
align-items: stretch;
 יישור הפריטים על הציר האנכי

מאפייני פריטי פלקסבוקס (Flex Items)

מאפיין דוגמה הסבר
flex-grow
flex-grow: 1;
 קובע את יחס הגדילה של הפריט
flex-shrink
flex-shrink: 0;
 קובע אם הפריט יתכווץ
flex-basis
flex-basis: 50px;
 קובע את הגודל הבסיסי של הפריט
