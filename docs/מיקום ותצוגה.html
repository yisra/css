מיקום ותצוגה (Position and Display)

מיקום (Position)

ערך דוגמה הסבר
static
position: static;
 המיקום ברירת המחדל, ללא מיקום מיוחד
relative
position: relative;
 מיקום יחסי למיקומו הרגיל
absolute
position: absolute;
 מיקום יחסית לאלמנט האב
fixed
position: fixed;
 מיקום קבוע ביחס לחלון הדפדפן
sticky
position: sticky;
 מיקום "דביק" המשתנה בהתאם לגלילה

סדר שכבות (Z-Index)

ערך דוגמה הסבר
מספר
z-index: 10;
 קובע את סדר הופעת האלמנט על ציר ה-Z
auto
z-index: auto;
 ברירת מחדל, ללא סדר מוגדר

תצוגה (Display)

ערך דוגמה הסבר
block
display: block;
 אלמנט קו-חדש (Block)
inline
display: inline;
 אלמנט בקו הטקסט (Inline)
flex
display: flex;
 תצוגת Flexbox
grid
display: grid;
 תצוגת Grid

נראות (Visibility)

ערך דוגמה הסבר
visible
visibility: visible;
 האלמנט נראה
hidden
visibility: hidden;
 האלמנט מוסתר אך תופס מקום
collapse
visibility: collapse;
 משמש לרוב בטבלאות, מסתיר ואינו תופס מקום
