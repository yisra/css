מעברים (Transitions)

מאפיין המעבר (Transition Property)

מאפיין דוגמה הסבר
transition-property
transition-property: background-color;
 קובע אילו מאפיינים ישתנו במעבר

משך המעבר (Transition Duration)

מאפיין דוגמה הסבר
transition-duration
transition-duration: 0.5s;
 משך הזמן של המעבר

פונקציית תזמון (Timing Function)

מאפיין דוגמה הסבר
transition-timing-function
transition-timing-function: ease-in-out;
 קובע את הקצב של המעבר

השהיית מעבר (Transition Delay)

מאפיין דוגמה הסבר
transition-delay
transition-delay: 0.3s;
 מגדיר השהיה לפני תחילת המעבר

אנימציות (Animations)

הגדרת Keyframes

Keyframes מאפשרים לקבוע שלבים שונים באנימציה:

@keyframes example { 0% { background-color: red; } 100% { background-color: yellow; } }

מאפייני אנימציה

מאפיין דוגמה הסבר
animation-name
animation-name: example;
 שם האנימציה כפי שהוגדר ב-Keyframes
animation-duration
animation-duration: 2s;
 משך הזמן של האנימציה
animation-timing-function
animation-timing-function: ease;
 פונקציית תזמון של האנימציה
animation-delay
animation-delay: 1s;
 השהיה לפני תחילת האנימציה
animation-iteration-count
animation-iteration-count: infinite;
 מספר הפעמים שהאנימציה תבוצע
animation-direction
animation-direction: alternate;
 כיוון האנימציה (קדימה/אחורה)
